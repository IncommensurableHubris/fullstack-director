# docs/ — legibility index

> **`docs/spec/` is declaration-truth** (the spine). Everything else **references** requirements by `REQ-NNN` and
> never restates their text.

- `spec/` — the spine (Constitution + REQ registry + capabilities + design-intent + architecture-constraints).
- `planning/` — the backlog ledger + sprint slices (references REQs).
- `design/` · `architecture/` · `quality/` — realizations (reference REQs).
