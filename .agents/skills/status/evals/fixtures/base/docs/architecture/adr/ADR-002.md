# ADR-002: Email magic-link sessions (no passwords, no third-party SSO)

- **Status:** Accepted
- **Satisfies:** Constitution §5; resolves AMD-002.

## Context
Auth must be passwordless with no third-party SSO (Constitution §5). REQ-020 reads are session-scoped to a team.

## Decision Outcome
Email **magic-link** one-time tokens minting a server-side session cookie; team scope enforced on every read.

## Consequences
- **Rule:** every digest read is authenticated + team-scoped. **Prevents** passwords and third-party SSO.
