# Design Intent

> Declared look/feel and key screens (a **declaration** — 02 may only *amend* this via the protocol). Owned by 00.

- **Tone:** calm, quiet, text-first — the digest is a document to read, not a dashboard to monitor.
- **Key screens:** the daily digest (read), the standup submission (write).
- **Accessibility:** WCAG AA — legible contrast, keyboard-navigable.
- **No notifications theatre:** "needs help" is surfaced in the digest and one channel post — nothing more.
