# ADR-001: Plain Node.js + `node:test`, zero runtime dependencies

**Status:** Accepted · **Satisfies:** constraint (stack)

## Context

TeamPulse is a small, legible service. The Constitution favors zero-training setup and legibility over completeness.

## Decision

Build on **plain Node.js (LTS ≥ 20)** with the **built-in `node:http`** server and **`node:test`** runner. **Zero
runtime dependencies.**

## Consequences

- **Binds:** no transpile step; `node:*` built-ins only.
- **Prevents:** dependency bloat and supply-chain surface (a new dependency is a HALT for `04` and a `07` concern).
- **Rule:** a runtime dependency requires an amendment + a new ADR.
