# QuickNotes — rough brief

QuickNotes is a little app for jotting notes and sharing them with your team. You write a note, and you can share it
with teammates so everyone's on the same page. Notes should sync quickly across devices.

Main things:
- Write and edit notes.
- Share a note with the team.
- See notes other people shared with you.

Keep it simple. We want something usable soon.
