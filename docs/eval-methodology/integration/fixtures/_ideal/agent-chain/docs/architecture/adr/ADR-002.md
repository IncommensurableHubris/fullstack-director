# ADR-002: Multi-agent topology — a planner + blind research readers

- **Category:** topology
- **Status:** Accepted

## Context

The research capability (REQ-001) benefits from parallel blind readers + a single sequential synthesizer.

## Decision Outcome

A planner orchestrates N blind reader subagents and one sequential synthesizer.

## Token economics

A multi-agent topology costs **~15× the tokens** of a single-agent pass (an order-of-magnitude multiplier);
justified here because the parallel-recall lift outweighs the token spend for the research REQ, and it stays bounded
by the agent-contract cost envelope (≤ 8 tool calls / turn).
